This is a Wayland-native screen recording script built around wf-recorder, designed to provide a simple TUI-based workflow instead of keybind-only recording.

Details 

→Record your entire screen, a specific monitor, or a selected region
→Optionally record system audio and/or microphone audio
→Control recording through an interactive menu
→Stop an ongoing recording using a floating TUI menu, without rerunning commands manually
→Automatically manage output directories and recording processes

Dependencies 

bash:Script runtime
wf-recorder:Core screen recording tool (Wayland)
fzf:Interactive TUI menus
slurp:Region selection on Wayland
libnotify:Desktop notifications
Required One:pipewire, wireplumber, pipewire-pulse

Dependencies Install 

pacman:sudo pacman -S --needed wf-recorder fzf slurp libnotify
yay:yay -S wf-recorder fzf slurp libnotify
